# Kelime Zinciri - Frontend Application

This directory contains the main application source code for **Kelime Zinciri**, a mobile word game built using React Native and Expo. 

Following recent updates, the game now operates entirely as a standalone client side application. Features requiring external server connectivity, such as user login and leadership leaderboards, have been completely decoupled. The full game architecture, UI components, and state management run directly within this frontend framework, ensuring a fast and stable local gaming experience.

*For complete project wide documentation, deployment guides, or general specifications, please refer to the main repository root README.*
